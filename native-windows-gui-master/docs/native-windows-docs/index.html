Native Windows GUI: The Guide

Welcome to Native Windows GUI (aka NWG). The BEST (and only) rust library to develop truly native GUI applications on desktop for Microsoft Windows.

This documentation is for NWG 1.0 . This is the 3rd and final version of NWG.

If you are new here jump to Getting started to install NWG and create your first application.

The API documentation is hosted on docs.rs https://docs.rs/native-windows-gui/1.0.0/native_windows_gui/
